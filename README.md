# 📢 Бот-Заклик для Telegram груп

## Встановлення

### 1. Отримай токен бота
- Напиши [@BotFather](https://t.me/BotFather) у Telegram
- Команда `/newbot`, задай ім'я та юзернейм
- Скопіюй токен (виглядає як `123456789:ABC-DEF...`)
- **Важливо:** у BotFather вимкни `Privacy Mode` (`/setprivacy → Disable`), щоб бот бачив всі повідомлення

### 2. Запуск локально (Python)

```bash
# Встанови залежності
pip install -r requirements.txt

# Запусти бота
export BOT_TOKEN="твій_токен_тут"
python bot.py
```

### 3. Запуск через Docker

```bash
docker build -t zaklyk-bot .

docker run -d \
  --name zaklyk-bot \
  --restart unless-stopped \
  -e BOT_TOKEN="твій_токен_тут" \
  -v zaklyk_data:/data \
  zaklyk-bot
```

## Команди

### 📢 Заклик
| Команда | Дія |
|---------|-----|
| `!заклик`, `!скликати`, `/call` | Скликати всіх учасників (кулдаун 1хв) |
| `/fix_call` | Перевірити заклик (тільки власник чату) |

Можна додати коментар: `!скликати хто ще не задонатив?`

### ✅ Реєстрація
| Команда | Дія |
|---------|-----|
| `!заклик+`, `заклик+`, `!call+`, `!прокинувся`, `!прокинулась`, `!з хлібом`, `/reg` | Повернутись до заклику в цьому чаті |
| `!заклик++`, `!call++`, `/reg_all` | Повернутись до заклику в усіх чатах |

### ❌ Вихід
| Команда | Дія |
|---------|-----|
| `!заклик-`, `заклик-`, `!call-`, `!оф`, `!піти по хліб`, `!тиха година`, `!я сплю`, `/unreg` | Вийти з заклику в цьому чаті |
| `!заклик--`, `/unreg_all` | Вийти з заклику в усіх чатах |

### 🔍 Інше
| Команда | Дія |
|---------|-----|
| `!заклик?` | Перевірити свій статус та смайлик |
| `!смайлик 🔥`, `/emoji 🔥`, `/set_emoji 🔥` | Змінити свій смайлик |
| `!перепис населення`, `/regchat` | Додати всіх до заклику (тільки власник) |

### ⚙️ Налаштування (тільки власник чату)
| Команда | Дія |
|---------|-----|
| `/call_del <сек>` | Автовидалення закликів через N секунд (0 = вимкнути) |
| `/call_admins_only` | Дозволити заклик тільки адмінам (перемикач) |

---

## Примітки

- Бот автоматично реєструє учасників, які пишуть у чат
- Telegram не дозволяє ботам отримати список усіх учасників без взаємодії з ними
- При `!заклик?` у відповідь на повідомлення — показує статус тієї людини
- SQLite база зберігається у файлі `bot.db` (або `/data/bot.db` в Docker)
